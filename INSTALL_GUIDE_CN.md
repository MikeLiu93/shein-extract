# SHEIN 上架工具 — 安装与使用指南

> 给员工看的简易版指南。Windows 10 / 11。

## 安装步骤（首次使用）

### 1. 装好这两个软件

**Google Chrome**
https://www.google.com/chrome/ — 选默认设置一直下一步即可。

**Google Drive for Desktop**
https://www.google.com/drive/download/ — 装完登录你那个有「02 希音」共享盘的 Google 账号，**等同步完成**（屏幕右下角图标停止转动）。

### 2. 双击安装包

`SheinExtract-Setup-3.5.x.exe`

按默认设置安装即可，会在桌面建一个【SHEIN 上架工具】图标。

### 3. 双击桌面图标启动

第一次启动会跳出**设置向导**，跟着走 5 步：

1. **欢迎** —— 直接点【开始】
2. **系统检查** —— 应该看到两个绿色 ✓
3. **路径设置** —— 4 个路径已经自动填好，看一眼对不对：
   - 输入 Excel 文件夹（共享盘里的 `submitted` 目录）
   - 输入 Excel 文件名（默认 `Shein Submited Links.xlsx`）
   - 输出根目录（共享盘里的 `Listing - completed 2nd`）
   - 备份目录 ⚠️ **这条要你自己选** —— 老板会告诉你备份云盘在哪
4. **登录 SHEIN** —— 点【打开 Chrome 登录 SHEIN】，在弹出的 Chrome 里登录你的卖家账号，登完直接关 Chrome，回向导点【已完成登录】
5. **测试** —— 点【开始测试】，看到"所有测试通过 ✓"就行

设置完成后向导会自动关闭。**以后每次双击桌面图标就直接开干**，不再走向导。

## 日常使用

### 跑上架

双击桌面【SHEIN 上架工具】 → 黑色控制台窗口出现 → 自动检查更新 → 开始抓取共享盘 Excel 里那些**还没填 Date+Status** 的行 → 跑完按 Enter 关窗口。

跑的过程中如果遇到验证码：
- 工具弹出的那个 Chrome 窗口里会显示 GeeTest 拼图
- 你**手动过一下**就行（点几下图标）
- 工具最多等 5 分钟，过了就跳到下一个 URL

### 合并总表

需要把某店铺的所有日表汇总成一个总表时：
- 双击桌面【合并总表】（如果有快捷方式），或者直接到工具安装目录运行 `merge_master.cmd`
- 输入店铺代号（比如 `C4`）回车
- 在 `共享云端硬盘\02 希音\Auto Pipeline\Listing - completed 2nd\C4\` 下生成 `C4_master.xlsx`

## 常见问题

### "未找到 Chrome"
还没装 Chrome，或装在了非默认路径。先装好 Chrome 再重新跑。

### "未在 D~K 盘下找到 共享云端硬盘"
Drive for Desktop 还没同步完，或挂在了奇怪的盘符上。等同步完再来；如果盘符不在 D~K，向导第 3 步手动填路径。

### "API 调用失败 HTTP 401"
共用 API key 失效了，找老板重新发一版安装包。

### Chrome 跑着跑着关了 / 不显示
关闭所有 Chrome 窗口（包括日常浏览的那个），再重启工具。工具会自己启一个独立的 Chrome 窗口。

### 备份目录路径错误
查向导第 3 步填的备份目录，确认共享备份盘已挂载。

### 改路径 / 重新设置
删除这个文件夹，再双击桌面图标会重走向导：
```
%APPDATA%\shein-extract\
```
（资源管理器地址栏粘贴 `%APPDATA%` 回车，找到 `shein-extract` 删掉）

### 自动更新
每次启动工具会**最多每 24 小时检查一次**新版本。有新版会弹窗问你要不要更新，点【立即更新】会自动下载替换重启。如果你点【本次跳过】，下次启动还会再问。
